SendPingRequestDirective

public struct SendPingRequestDirective

Describes how a pingRequest should be performed.

Only a single target peer is used, however it may be pinged “through” a few other members. The amount of fan-out in pingRequests is configurable by swim.indirectProbeCount.

    Target that the should be probed by the requestDetails.memberToPingRequestThrough peers.
